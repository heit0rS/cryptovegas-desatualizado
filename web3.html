As fases da internet


Para compreendermos o que é o Web 3.0, temos que aprender que existem antecessores do mesmo, sendo eles o Web 1.0 e o Web 2.0.

As diferentes numerações da Web (1.0; 2.0; 3.0) podem ser definidas como as principais fases da evolução e construção da internet que conhecemos atualmente. Logicamente, a Web 1.0 é a fase inicial da internet, onde começa a ser disponibilizada para todos, a Web 2.0 já é uma fase mais trabalhada em funcionalidades para os usuários e uma maior facilidade de acesso e uso da internet. A Web 3.0 é a internet como conhecemos hoje em dia, sendo acessível a todos e em qualquer lugar e dispositivo, cheia de infinitas possibilidades de uso e sem um órgão principal responsável, ou seja, descentralizada.



Web 1.0


A web 1.0 pode ser chamada de “o mundo novo”, é a fase que se estende pela década de 1990, com conteúdo não animados e com pouca ou nenhuma interação com outros usuários, assim a web 1.0 foi utilizada predominantemente em ambientes acadêmicos e de pesquisa, a pessoa que iria utilizar ficava off-line.
